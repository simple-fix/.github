#### 👋 Open source, from day one.

<sub>simple-fix was founded by **Ahed Eid (عاهد عيد)** with a simple belief: clients should start their work using and creating open‑source projects. We keep our tools public, our code clean, and our process transparent. Whether you need a high‑converting landing page or a complex real‑time dashboard, we build it in the open. [Learn more about Ahed →](https://ahed91.simple-fix.site)</sub>

#### 🦺 Keeping things simple—and safe.

<sub>The simple-fix GitHub org follows a common‑sense Code of Conduct. Be respectful, be helpful, and keep the vibes collaborative. When interacting with any repository under this org, please treat others the way you'd want your own code reviewed.</sub>

#### 👾 Reporting security issues.

<sub>Found a vulnerability? Please don't open a public issue. Instead, email us at **security@simple-fix.site** and we'll address it promptly. We respect confidentiality and will credit you for responsible disclosure.</sub>
